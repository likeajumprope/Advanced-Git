Command Index

Every command taught in Advanced_Git, organized by topic.

A complete reference for all terminal commands and Git operations taught across Modules 00–12. Commands are organized by topic — use your browser’s Ctrl+F / Cmd+F to search for a specific command.

Navigation & Environment

Commands for moving around the filesystem and inspecting your environment.

Command Description Key Flags Module
pwd Print current working directory — 00
ls List directory contents -la (all + details), -1 (one per line) 00
cd PATH Change directory .. (parent), ~ (home), - (previous) 00
mkdir NAME Create a directory -p (create parents) 00
touch FILE Create empty file or update timestamp — 00
cat FILE Display file contents -n (line numbers) 00
rm FILE Remove a file -r (recursive), -f (force) 00
mv SRC DEST Move or rename a file — 00
cp SRC DEST Copy a file -r (recursive) 00
nano FILE Open file in nano editor — 00
COMMAND --help Show command help — 00
man COMMAND Open manual page — 00
history Show command history \| grep PATTERN (filter) 11
tree DIR Show directory tree — 12

Repository Init & Config

Setting up repositories and configuring Git identity.

Command Description Key Flags Module
git init Initialize a new repository — 01
git config user.name "Name" Set commit author name --global (all repos) 01
git config user.email "email" Set commit author email --global (all repos) 01
git config init.defaultBranch main Set default branch name --global 01
git config --list Show all configuration --global, --local 01
cat .git/HEAD Show current branch ref — 01
cat .git/config Show repository config — 01
git config --global alias.NAME "cmd" Create a command alias — 11

Staging & Committing

The add-commit workflow for recording changes.

Command Description Key Flags Module
git status Show working tree status -s (short format) 02
git add FILE Stage a file . (all), -p (interactive patch) 02
git restore --staged FILE Unstage a file — 02
git restore FILE Discard working directory changes --source HASH (from commit) 10
git commit -m "message" Commit staged changes --amend (modify last commit) 02
git diff Unstaged changes --staged (staged), HASH..HASH (between commits) 02
git rm --cached FILE Stop tracking a file (keep on disk) — 08
git check-ignore -v FILE Explain why a file is ignored — 08

Command Description Key Flags Module

git log -S "string" Commits adding/removing string — 07
git log --grep "pattern" Commits matching message pattern -i (case-insensitive) 07
git show HASH Show commit diff and metadata --stat (summary only) 07
git blame FILE Line-by-line last-modification info -L N,M (line range), -w (ignore whitespace) 07
git reflog Show HEAD movement history — 10
git ls-files List all tracked files — 08
git count-objects -vH Repository size breakdown — 08
git cat-file -p HASH Show raw object content -t (type only) 01
git fsck Check repository integrity --lost-found 10

Branching

Creating, switching, and managing branches.

Command Description Key Flags Module
git branch List local branches -v (with last commit), -a (include remotes), --merged 04
git switch BRANCH Switch to a branch -c NAME (create and switch) 04
git checkout BRANCH Switch to a branch (legacy) -b NAME (create and switch) 04
git checkout HASH Detach HEAD at a commit — 10
git branch -d BRANCH Delete a merged branch -D (force delete) 04
cat .git/refs/heads/BRANCH Show commit hash for branch — 04

Combining and rewriting history.

Command Description Key Flags Module

git merge --abort Abort an in-progress merge — 05
git rebase BRANCH Rebase current branch onto target — 06
git rebase -i HEAD~N Interactive rebase of last N commits — 06
git rebase --continue Continue after resolving rebase conflict — 06
git rebase --abort Abort an in-progress rebase — 06
git cherry-pick HASH Copy a specific commit to current branch — 07
